# YaCUT
Сервис для сокращения URL и загрузки файлов на Яндекс Диск с генерацией коротких ссылок.

### Функционал
Сокращение длинных URL до коротких идентификаторов (например, yoursite.com/abc123).

Загрузка файлов через веб‑форму с сохранением на Яндекс Диск.

Генерация коротких ссылок для прямых загрузок файлов.

Поддержка пользовательских коротких ID (при доступности).

Веб‑интерфейс для обеих операций.

API для создания коротких ссылок программно.

### Технологии
-Python 3.8+

-Flask — веб‑фреймворк

-SQLAlchemy — ORM для работы с БД

-aiohttp — асинхронные HTTP‑запросы к API Яндекс Диска

-WTForms — формы и валидация

-Яндекс Диск API — хранение загруженных файлов

### Установка и настройка
Предварительные требования
-Python 3.8 или выше

-Яндекс Диск (аккаунт и OAuth‑токен)

-PostgreSQL или SQLite (для хранения ссылок)

### СУБД:

для SQLite: ничего дополнительно настраивать не нужно — файл БД будет создан автоматически в директории проекта.

### Как запустить проект Yacut:

Клонировать репозиторий и перейти в него в командной строке:

```
git clone git@github.com:DmitryTre/async-yacut.git
```

### Cоздать и активировать виртуальное окружение:

```
cd async-yacut
```

```
python3 -m venv venv
```

* Если у вас Linux/macOS

    ```
    source venv/bin/activate
    ```

* Если у вас windows

    ```
    source venv/scripts/activate
    ```

Установить зависимости из файла requirements.txt:

```
python3 -m pip install --upgrade pip
```

```
pip install -r requirements.txt
```

Создать в директории проекта файл .env с четыремя переменными окружения:

```
FLASK_APP=yacut
FLASK_ENV=development
SECRET_KEY=your_secret_key
DB=sqlite:///db.sqlite3
```

Создать базу данных и применить миграции:

```
flask db upgrade
```

Запустить проект:

```
flask run
```
